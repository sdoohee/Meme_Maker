# Meme_Maker
> 바닐라JS로 그림 앱을 제공하는 사이트입니다.


기본적으로 그립판에 들어있는 기능들을 제공합니다.

추가 : 이제는 '짤 생성기' (Meme Maker. 밈 메이커)를 만들자!

![](../header.png)

## 구현하는 기능

핵심:

- HTML5 Canvas
- Mouse Events
- 2D Context
- 2D Painting
- Brush Size
- Image Saving

기술:

- HTML
- CSS
- JavaScript
